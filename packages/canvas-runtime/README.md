# @ecs-editor/canvas-runtime

Game loop, canvas wrapper, input handling, and timing utilities.

## Features

- **Canvas2D**: Wrapper around HTML5 Canvas 2D rendering context
- **GameLoop**: Fixed timestep game loop with system execution
- **InputManager**: Keyboard and mouse input handling
- **Timer**: Timing utilities for game logic

## Usage

```typescript
import { Canvas2D, GameLoop, InputManager } from '@ecs-editor/canvas-runtime';
```
