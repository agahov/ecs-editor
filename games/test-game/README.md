# Test Game

Simple game that moves several entities on screen.

## Structure

- `config/` - Game configuration
- `init/` - Initialization code
- `aspects/` - Custom game-specific aspects
- `systems/` - Custom game-specific systems
- `levels/` - Level data and entity loading
- `ui/` - Game UI
- `assets/` - Game assets

## Running

```bash
npm run dev
```
